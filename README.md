# Pricing Service

Minimal Python pricing service (small example project).

## Requirements

- Python 3.8+
- See `requirements.txt` for Python packages

## Setup

Create a virtual environment and install dependencies:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

## Run tests

```bash
pytest
```

## Project layout

- `app/` — service code
- `tests/` — tests
